# Raid Law Glossary

Shared language for Platinum DKP.

```text
We are the fire that binds.
```

Meaning:

- the community holds the line together
- receipts carry memory across the raid
- the founder starts the fire, but public rules keep it from becoming smoke
- no one gets to turn heat into hidden authority

## Core Translations

| Plain term | Raid-law term | Meaning |
| --- | --- | --- |
| creator | raid founder | origin authority, not secret override authority |
| rules | raid law | public operating contract |
| DKP | honor ledger | off-chain contribution reputation |
| token | banner | public symbol, not the game itself |
| agent | scribe | formats, checks, refuses, and records |
| receipt | loot log | durable proof of action and decision |
| treasury | guild bank | slow, governed, public custody |
| proposal | raid motion | written request for review and vote |
| market talk | auction tunnel | risky zone; no pump commands |
| paid action | paid scribe service | service request only, not power |
| moderation | raid marshal lane | protects the table and the receipts |
| patch note | patch day law | public rule changes |

## Dragon Slayer Oath

We slay confusion, scams, drama, fake urgency, seller-shaming, and whale aggro.

We do not slay dissent.

We do not sell the loot table.

We do not call market movement a quest.

We do not confuse a payment with a vote.

The raid survives because the rules are public before the boss dies.

## Style Rule

Raid language can make the system memorable. It cannot change the contract.

If a joke conflicts with a rule, the rule wins.

If a metaphor hides risk, rewrite it plainly.

If money, treasury, custody, market behavior, or eligibility is involved, state the literal rule first and the joke second.
