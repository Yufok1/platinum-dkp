# Appeals And Patch Notes

## Appeals

Appeals exist because sometimes the loot council needs to read the actual rule instead of vibes in a robe.

Appeal form:

```text
Appeal title:
Original decision:
Original receipt:
Rule involved:
New evidence:
Requested correction:
Why this matters:
```

Appeal outcomes:

- uphold
- partial correction
- reverse
- needs vote
- more evidence required

## Patch Notes

Rule changes must be visible.

Patch note form:

```text
Patch version:
Date:
Changed rule:
Old text:
New text:
Reason:
Effective date:
Retroactive:
Affected quests:
Migration note:
```

## Versioning

Use simple versions:

```text
v0.1 initial field rules
v0.2 quest scoring update
v0.3 treasury gate clarification
```

Never silently change rules after a dispute starts. That is how tabletop arguments become archaeology.
