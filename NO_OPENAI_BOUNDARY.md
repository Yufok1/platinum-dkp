- Hugging Face Spaces
- local model wrappers
- repo documentation
- community rulebooks
- Discord moderation scripts
- human-run governance

## Secret Handling

Do not paste secrets into model chats.

Do not upload:

- seed phrases
- private keys
- wallet credentials
- exchange credentials
- API secrets
- private treasury notes

If a secret is exposed, rotate it. Do not ask the model to remember it.
