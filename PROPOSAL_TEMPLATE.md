# Proposal Template

Use this for governance, rule changes, guild bank requests, reviewer appointments, and major quest expansions.

```text
We are the fire that binds.
```

```text

Recipient, if any:

Conflicts of interest:

Risks:

Abuse cases:

Review window:

Vote window:

Quorum:

Execution delay:

Receipt plan:

Rollback plan:
```

## Proposal Types

- rule change
- new quest line
- guild bank request
- reviewer appointment
- moderation escalation
- appeal decision
- public communication

## Required Classification

Every proposal must be classified:

- `allowed`
- `needs_vote`
- `risky`
- `rejected`

Treasury requests should normally classify as `needs_vote`.
