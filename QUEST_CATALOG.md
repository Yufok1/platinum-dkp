# Quest Catalog

Quests are raid work. The point is useful contribution with proof, not market noise.

```text
We are the fire that binds.
```

## Quest Template

```text
Quest name:
Dragon type:
Objective:
Proof required:
DKP range:
Cooldown:
Abuse cases:
Reviewer:
Receipt link:
```

## Dragon Types

The dragon taxonomy names the problem class before the raid swings at it.

### FUD Dragon

Confusion, rumor, or panic answered with public receipts.

Example quests:

- write a neutral explainer for a repeated question
- summarize a source without hype
- collect links that resolve a false claim

Default DKP: 5-20

### Scam Dragon

Fake links, impersonators, spoofed accounts, and malicious instructions.

Example quests:

- report an impersonator
- verify official links
- write a safe-wallet hygiene note
- archive scam evidence for moderators

Default DKP: 10-40

### Lore Dragon

Memes, explainers, art, myth, and culture that do not imply profit.

Example quests:

- write a lore card
- design a safe meme
- make a glossary
- produce a non-market onboarding joke

Default DKP: 5-25

### Onboarding Dragon

Helping newcomers understand the game safely.

Example quests:

- answer a basic question with no price pressure
- guide someone to official docs
- explain DKP and quests
- write beginner instructions

Default DKP: 5-30

### Archive Dragon

Screenshots, links, votes, receipts, and durable memory.

Example quests:

- archive a governance vote
- compile patch notes
- preserve a proposal thread
- verify a receipt trail

Default DKP: 10-35

### Chaos Dragon

Drama de-escalation and room cooling.

Example quests:

- summarize a dispute neutrally
- identify the actual rule question
- propose a fair appeal process
- move market panic into a rule lookup

Default DKP: 10-30

### Whale Aggro Dragon

High-capital entitlement trying to become governance.

Example quests:

- identify a private-privilege request
- redirect a whale demand into public proposal process
- document unequal-access risk
- write a governance-safe response

Default DKP: 15-50

## Starter Quests

### Quest: First Receipt

Dragon type: Archive Dragon

Objective: create one public receipt for a completed contribution.

Proof required: link to contribution and one-sentence summary.

DKP range: 5-10

Cooldown: once per user.

Abuse cases: fake link, duplicate work, self-review.

Reviewer: any approved reviewer.

### Quest: Noob Shield

Dragon type: Onboarding Dragon

Objective: help one newcomer understand DKP without market advice.

Proof required: screenshot or link to answer.

DKP range: 5-15

Cooldown: once per day.

Abuse cases: price talk, wallet pressure, spam.

Reviewer: onboarding reviewer.

### Quest: Scam Bonk Receipt

Dragon type: Scam Dragon

Objective: identify and archive a suspicious link, impersonator, or fake claim.

Proof required: screenshot, URL, why it is suspicious, and report destination.

DKP range: 10-40

Cooldown: no cooldown for high-risk active scams.

Abuse cases: false accusation, doxxing, repeated low-signal spam.

Reviewer: moderation reviewer.

### Quest: Patch Note Scribe

Dragon type: Archive Dragon

Objective: turn a rule change into readable patch notes.

Proof required: before rule, after rule, reason, effective date.

DKP range: 10-25

Cooldown: per patch.

Abuse cases: editorializing, hiding risk, omitting backward compatibility.

Reviewer: governance reviewer.

### Quest: Whale Aggro Redirect

Dragon type: Whale Aggro Dragon

Objective: redirect a private privilege demand into public governance.

Proof required: sanitized quote or summary plus proposed public process.

DKP range: 15-35

Cooldown: once per incident.

Abuse cases: harassment, exposing private identity, escalating conflict.

Reviewer: governance moderator.
