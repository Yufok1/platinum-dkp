# DKP Scoring Table

DKP is reputation for contribution. It is not a claim on tokens, treasury, returns, or market access.

## Bands

| Band | DKP | Use |
| --- | ---: | --- |
| Pebble | 1-5 | Tiny useful action, typo fix, small answer |
| Bronze | 5-15 | Basic quest, newcomer help, small receipt |
| Silver | 15-30 | Good contribution with proof |
| Gold | 30-60 | High-signal work, scam defense, major doc pass |
| Platinum | 60-100 | Rare, durable contribution with strong evidence |

## Multipliers

Only use multipliers when stated before the quest starts.

Allowed:

- first-time public documentation
- verified scam prevention
- high-quality onboarding
- governance archive completion
- difficult de-escalation

Rejected:

- holder status
- purchase size
- trading volume
- social follower count alone
- insider access
- private whale pressure

## Review Checklist

Before awarding DKP:

1. Is the quest valid?
2. Is proof present?
3. Is proof public or reviewable?
4. Is the work original enough?
5. Is the work non-market-manipulative?
6. Is the cooldown clear?
7. Is a receipt recorded?

## Duplicate Work

If two users do the same job:

- first complete receipt gets normal scoring
- second may get partial DKP if it adds value
- copy-paste duplicates get zero
- bad-faith duplicates may trigger moderation

## Reviewer Limits

Reviewers cannot score their own submissions.

Large awards should require two reviewers.

Suggested threshold:

```text
50+ DKP requires two reviewers
100+ DKP requires governance note
```
